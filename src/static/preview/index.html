借鉴 Saul Bass 减法、Paul Rand 符号化、原研哉空白的张力、Gestalt 一笔双关

L1 生物神经 = 弧线 + 分支 + 圆节点

4条主弧线围合成大脑半球轮廓（一笔双关）
6个圆形突触 · 无直线无直角

L6 融合态 = 左弧线 + 右折线 + 中线交汇

左侧2条贝塞尔弧线 vs 右侧2条45°PCB走线
左圆节点 vs 右方过孔 · 中心融合光核

L8 超越态 = 纯弧对称 + 环流 + 溢出

3层同心弧线环流 · 无任何直角
冠冕弧跨左右半球 · 溢出射线

设计铁律

负空间就是内容
