# 段位徽章 v6 设计方案

## 问题诊断（v5 → v6）

v5 核心概念正确，但执行层面犯了"画图思维"错误：
- 大脑轮廓用连续路径描摹 → 像解剖图，不像徽章
- 神经网络 4层×6分支 = 24条线 → 杂乱无章
- PCB 线路混在弧线中 → 看不出来
- 整体缺乏"经过设计的克制感"

## v6 核心方法论：符号化极简

**总原则：用 3-6 条线传达 L1/L6/L8 三种状态**

借鉴的设计哲学：
| 大师 | 核心理念 | 对 v6 的指导 |
|------|---------|-------------|
| Saul Bass | 提炼到无法再减 | 每条线必须承载含义，砍掉所有装饰线 |
| Paul Rand | 符号不是画，是"写" | 用最少的几何笔画书写"神经"二字 |
| 原研哉 | 空白的张力 | 负空间（两半球间隙）本身就是内容 |
| Gestalt 理论 | 整体大于部分之和 | 神经分支的弧线外沿=大脑轮廓，一笔双关 |

## 三大状态的可视化符号系统

### 生物神经网络（L1）= 弧线 + 分支 + 圆节点
识别特征：光滑贝塞尔曲线、Y形分叉、圆形突触
**极简版本：4条主弧线 + 中央桥 + 正中裂隙 + 6个节点**
→ 弧线围合的形状即大脑半球轮廓（一笔双关）

### 硅基电路（L6 右半）= 45°折线 + 平行线 + 方形过孔
识别特征：精确45°/90°转角、平行走线、方形焊盘
**极简版本：左2条弧线 + 右2条45°折线 + 圆/方节点混合**
→ 左右对比一目了然，中线处交汇

### 超越态神经网络（L8）= 完美弧线 + 对称 + 光环
识别特征：纯弧线（无直线无折角）、同心环流、双侧完美对称
**极简版本：2层环流弧线 + 中央光核 + 冠冕弧 + 4条溢出射线**
→ 像神经曼荼罗，比 L1 更密但更有秩序

## 8 段位渐进叙事

```
L1  纯生物   4弧线 6节点  —— 混沌初开
L2  纯生物   5弧线 8节点  —— 枝蔓生长
L3  生物主导 5弧线+1直角  —— 技术初触
L4  半融合   4弧线+2直角  —— 交织加速
L5  技术主导 3弧线+3直角  —— 硅基觉醒
L6  均衡融合 2弧线+2直角  —— 嬗变临界（炼金术士）
L7  回归弧线 4弧线+1直角  —— 硅基消融
L8  纯弧超越 8弧线 12节点 —— 人机无界
```

## 帧结构与材质（保持 v5 已验证的体系）

每枚徽章：外光晕 → 装饰环 → 金属边框渐变 → 暗色底盘 → 中心符号 → 表面高光 → 粒子
边框厚度：L1=5px, L6=7px, L8=8px
粒子数量：L1=3, L6=9, L8=16
